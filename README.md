# node-case

- [x] 🔨 chat-room(实现多人聊天室功能)
- [x] 🔨 elk(写入日志的 Elk 演示)
- [x] 🔨 express-swagger(生成 swagger api 文档)
- [x] 🔨 fileupload(大文件分片上传)
- [x] 🔨 video-stream(视频流播放)
- [x] 🔨 crypto(encryption and decryption)
